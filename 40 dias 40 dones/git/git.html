GIT BASICS

git initcreates an empty Git repository in specified directory. Runs directly the current directory as a git repository

git cloneClone repo located at "repo" into local machine. Original repo can be located on the local filesystem or on a remote machine via HTTP or SSH.

git config user.name define author name to be used for all commits in current repo. Devs commonly use --global flag to set config options for current user.
